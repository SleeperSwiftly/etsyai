# EtsyDesc AI

AI-powered Etsy product description generator. Generates 3 SEO-optimized descriptions in seconds.

## Setup

1. Clone this repo
2. Run `npm install`
3. Copy `.env.local.example` to `.env.local` and add your Anthropic API key
4. Run `npm run dev` to start locally

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your GitHub repo
3. Add environment variable: `ANTHROPIC_API_KEY` = your key
4. Deploy!
